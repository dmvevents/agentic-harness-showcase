Capabilities

Every capability below is tagged with an honest status: shipped works today, planned v2 is designed but not built, gated is wired but held behind a human gate or booked capacity, and partial works one way but not yet another. No capability here is aspirational marketing — if it isn't shipped, it says so.

Core capabilities

Three lanes, in depth

The agent exposes exactly three verbs. Each routes to deterministic tool functions — the language model fills slots and formats output, it never computes the answer.

ask

Knowledge Q&A shipped

Routes a HyperPod / EKS / EFA question to a curated, version-controlled knowledge base. Returns a do / do-not / depends answer with a source citation and a confidence — never an unsourced guess.

  • 30-entry HyperPod Q&A KB, PR-gated
  • answers carry a source + confidence
  • refuses rather than hallucinates when off-corpus
capacity

Capacity discovery shipped

Read-only discovery of EC2 Capacity Block and SageMaker Flexible Training Plan offerings — instance type, count, region, duration, price, start window. The same call our team uses to answer real "can we get N GPUs by DATE" asks.

  • EC2 Capacity Blocks (no allowlist)
  • FTP offerings (allowlist-gated path detected + reported)
  • multi-region, multi-instance sweep
  • purchase is separate — see the firewall below
orchestrate

Plan & remediate gated

A goal sentence becomes an ordered module plan, then the empirical ladder: apply → observe → classify the failure → match an instinct → dispatch the codified fix → retry. Plan synthesis catches hardware traps before any apply.

  • plan synthesis: shipped
  • stage dry-run mode: shipped
  • live multi-node bring-up: gated on booked capacity

The differentiator

A failure classifier, not a chatbot

Paste a log signature; get the same instinct, the same severity, and the same codified fix every time. 52 compiled patterns, no model in the loop. Severity decides whether a benchmark number is even trustworthy.

SeverityMeaningPatternsExample
BLOCKrun is invalid — stop, the result is meaningless10EFA device not present; IAM denies the action
DROPsilent correctness loss — packets / collectives failing17SRD on a secondary CIDR being silently dropped
DEGRADEworks but slow — the number understates real capability18NCCL fell back to TCP sockets; bandwidth is bogus
WARNworth noting, not fatal7suboptimal NCCL algo / protocol selection
Why severity matters for a benchmark portal. A bandwidth number captured while NCCL was silently on TCP (DEGRADE) isn't slow — it's wrong. The classifier flags that before the number ever reaches a customer report.

What it deliberately will not do

The capabilities behind a human gate

The most important capability is a refusal. Spending money and mutating a cluster are deliberately not autonomous.

spend firewall

Cannot buy capacity on its own

A capacity purchase is non-refundable, so it sits behind a 3-factor gate the agent cannot satisfy from its own outputs:

  • CLUSTER_OPS_PROD=1 — the live-cluster env gate
  • --confirm must equal the exact offering id
  • an out-of-band CLUSTER_OPS_PURCHASE_TOKEN a human sets — never emitted by any tool, so the agent has nothing to copy
non-mutating surface

Cannot mutate over the public MCP

The inbound MCP server exposes only read-only verbs (ask, classify, plan, discover, taxonomy). The mutating verbs (purchase, fix-execute) live in-process, never on the MCP catalog.

  • enforced by a build-failing test, not a convention
  • add a purchase verb to the MCP catalog → CI goes red

Benchmark intelligence

A source-of-truth benchmark repo + deterministic report tools

source of truth

PR-gated benchmark repo

Schema'd artifacts (config + results + provenance + validation status), a publication state machine (draft → validated → published), and a CI gate that rejects unreviewed or sensitive data. Every metric is tagged measured or estimated.

deterministic tools

query_benchmark · generate_report

Read-only tools the agent calls to find artifacts and assemble a customer-safe report. A number-provenance guard raises if any number in the report can't be traced to a tool result — the agent structurally cannot fabricate a benchmark figure. See the benchmark portal →
